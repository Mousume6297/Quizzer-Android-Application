# Quizzer Android Application

Quizzer is an Android-based quiz application developed to test and improve users' knowledge through multiple quiz categories and difficulty levels. The application provides an interactive and user-friendly interface for answering questions and viewing results.

## Features

- Multiple quiz categories
- Different difficulty levels
- Secure user login and registration using Firebase Authentication
- Dynamic quiz questions using Firebase Realtime Database
- Automatic score calculation and result display
- Smooth navigation between activities
- Responsive and user-friendly UI

## Technologies Used

- Java
- XML
- Firebase Authentication
- Firebase Realtime Database
- Android Studio
- Object-Oriented Programming (OOP)

## Key Concepts Implemented

- Object-Oriented Programming (Encapsulation, Abstraction, Inheritance)
- Exception Handling
- Intents for navigation between activities
- Event Listeners for user interactions
- Collections Framework (ArrayList) for managing quiz questions
- Firebase integration for authentication and database management

## How the Application Works

1. User registers or logs in using Firebase Authentication.
2. User selects a quiz category and difficulty level.
3. Questions are retrieved from Firebase Realtime Database.
4. User answers questions and navigates through the quiz.
5. The application calculates the score and displays the result at the end.
